github部署

如果以上的网页能正常显示，那么恭喜你，本地环境搭建已经基本完成，现在是时候将hexo部署到github上了，将hexo部署你还需要完成几个步骤：

  1. 创建一个github账号
  2. 创建一个新的软件仓库，名称格式必须为：github的用户名.github.io。

    如：我的github用户名为：AriaLyy，那么该软件仓库的名必须为: AriaLyy.github.io
